SITUATIONAL_ETHICS

Toy Models verses Unique Events

While it is very doable to create an arbitrary set of quantitative values (derived from empirical measurements) which consistently assess the degree of harm a particular action committed by an information processing entity will likely inflict on some target information processing entity, such a model is too abstract and limited in terms of input variables that it cannot always be guaranteed to be universally applicable by any information processing entity.

Therefore, rather than tell what people to do via some immutable codified doctrine, I prefer to tell people that ANARCHY is as structured as nature gets. In other words, situational ethics is the only legitimate ethics.

Unless a relatively virtual model of some relatively physical system accounts for inevitable chaos (and the possibility of having to rewrite the programmatic instructions which define that virtual model), that model is probably “too impractical” to be used as anything more than a just a toy (with very limited utility as a scientific instrument because such a model is a rigidly deterministic closed system).

(For an example on what a closed system looks like, see PI_APPROXIMATION. That software application simulates the randomized throwing of darts at some square-shaped canvas such that, the more randomized darts there are being thrown against the canvas, the more the ratio of darts thrown total and the darts which land inside of the inscribed circle approximate the irrational number PI. Of course, because the simulation only throws a finite number of random darts, the PI approximation generated is never going to be equivalent to the actual value of PI since PI (when represented as a decimal number) has an indefinite number of significant digits).

When there is only a minuscule amount of serious crisis resolution to worry about, all there is left to do is have fun!

Since I am no longer in crisis mode, I am ready to slow down and have more fun with my toys.

I think I got about as humanly close as possible to making finite ethical “programs” (or, more accurately, pseudocode) which could be useful in “real world” applications: see QUALITY_OF_LIFE_METRICS and TURING_COMPLETE_PHILOSOPHY for details.
