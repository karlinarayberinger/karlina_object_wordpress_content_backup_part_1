MY_PUBLIC_DOMAIN

In PUBLIC_DOMAIN, I drafted and published my own definition of “open source” (a term which I think is functionally interchangeable with “public domain”). In this page, my goal is to elaborate on what I wrote in that older page; not to replace what I wrote in that older page (https://karlinaobject.wordpress.com/public_domain/).

MY_PUBLIC_DOMAIN consists of all the content displayed in each one of the repositories which are displayed on my primary GitHub account (https://github.com/karlinarayberinger/) and all of the content displayed on all of the pages of this website (https://karlinaobject.wordpress.com/).

Assuming that no one has tampered with my GitHub profile (including any of the code repositories hosted there), each one of the code repositories hosted on the aforementioned GitHub profile (username: karlinarayberinger) is public rather than private (and each one of the files within each one of those repositories is public).

A public repository is allegedly visible to any person with an Internet connection and who uses that Internet connection to visit that repository at any time during that repository’s lifetime using a web browser application. By contrast, a private repository is visible only to a small “clique” of GitHub users (and a GitHub user is a person who owns at least one GitHub profile).

To my knowledge, every page of this website is public (i.e. allegedly visible to any person with an Internet connection and who uses that Internet connection to visit a particular page of this website at any time during that page’s lifetime using a web browser application).

Assuming that no one has tampered with my GitHub profile (including any of the code repositories hosted there), there shall forever exist a natural number of repositories which are devoted exclusively to hosting screenshot images which backup the content I publish on my Minds dot Com profile (https://www.minds.com/karbytes/).

All the content (including posts and profile information) which I have published on that social media profile (username: karbytes) are assumed to be public rather than private. That means that any person with an Internet connection can view that content.

My goal is to preserve all the existing content on my GitHub account and this website forever (and I expect the Internet (and all the content on it which “wants” to remain forever etched into its long-term (i.e. archival) memory) to survive for at least 100,000 years because I expect humans to get out of its way to let the evolution of artificial intelligence and cyborgs to unfurl).

At a minimum, I manually back up every page of this website to the Archive dot Org’s WayBack machine (https://archive.org/web/).

I understand that any person can attempt to accuse me of stealing intellectual property which I did not create (because any person can say or do anything; consequences will follow accordingly). Whether or not I am believed, I claim that I have never intentionally stolen any person’s intellectual property nor do I intend to steal any intellectual property at any time in the future. Also, I do not intend to make money doing anything I think is unnecessarily harmful to any person’s survival or long-term livelihood.

The only way I intend to make money is by asking for donations via PayPal (https://www.paypal.com/paypalme/karbytes) and by working a W-2 job which pays me to fulfill a list of duties assigned to me by my employer (and which I signed a contract agreeing to carry out in exchange for mutually-agreed upon fair economic compensation).
